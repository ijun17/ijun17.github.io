# 스택

## 특징
* First In Last Out
* 추가, 삭제, 확인 : o(1)
* top 원소만 확인 가능

## STL stack
* push, pop, top : O(1)

## 스택 사용 문제
* 괄호쌍 문제
* 오래된 데이터는 오래 저장하고, 최근 원소는 많은 수정을 할 때

# 큐

## 특징
* First In First Out
* 추가, 삭제, 확인 : o(1)
* front 원소만 확인 가능

## STL stack
* push, pop, front : O(1)

## 스택 사용 문제
* 괄호쌍 문제
* 최근 데이터는 오래 저장하고, 오래된 원소는 많은 수정을 할 때
