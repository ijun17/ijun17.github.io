* `shutdown -h now` : 바로 종료
* `rm -rf 이름` : 강제로 지우는것, 복구불가

ls
* `ls -l` : 기본 파일들만
* `ls -als` : 숨김 파일 포함 모든 파일 리스트
* `ls -als > test` : test 파일에 결과 텍스트로 저장

mv
* `mv 기존파일이름 새로운파일이름`


vi
* `vi test` : test 파일 편집
* 입력모드 : `i`
* 명령모드 : esc
* 라인모드 : `:set nu`
* 방향 키 : h, j(아래), k(위), l
* 한행지우기 : `dd`
* 저장 나가기 : `:wq`
* 그냥 나가기 : `:q`
* 저장안하고 나가기: `:q!`
* ^
* &
* /pattern
